Function Documentation

void ioe_init ( )

Initializes the MCP23S17 I/O expander and configures one of the PIC's SPI modules for communication with the I/O expander.

Pin mappings: RPB5 (pin 14) –> SDO2 (MOSI) RPA4 (pin 12) –> SDI2 (MISO)

Uses RB9 (pin 18) as the chip select (CS) line.
