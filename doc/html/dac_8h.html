Detailed Description

A PIC32 library for the MCP4822 dual 12-bit DAC.

Intended for use with the PIC32MX250F128B.

Author
Jeff Lutgen (based on code by Bruce R. Land)

Function Documentation

void dac_init ( )

Configures and enables a SPI channel and CS line for communicating with the DAC.

Pins used: 

CS:         RB4  (pin 11)
SCK:        SCK2 (pin 26)
SDO (MOSI): RPB5 (pin 14) --> SDO2
void dac_write ( uint16_t  msg)
inline

Writes a 16-bit word to the DAC.

Upper 4 bits: configuration bits
Lower 12 bits: data

Example: 

// 0 <= my_data < 4096
dac_write(DAC_A | DAC_GAIN1X | DAC_ACTIVE | my_data);
